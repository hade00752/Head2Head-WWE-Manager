# Head2Head-WWE-Manager
Downloads for Head 2 Head Wrestling Manager, an unofficial WWE fan-made GM sim. Binaries only, source is not published here.

  A free, database-driven wrestling GM simulator. Built with the aid of Claude Code.
  
  This repository hosts downloads only. No source code is published here.

  ## Download
  
  - Windows: [Releases](../../releases/latest)
  - Linux: [Releases](../../releases/latest)
  
  ## How to Run

  1. Download the zip for your OS from Releases.
  2. Extract it anywhere.

  **Windows:**
  - Run as administrator: `H2H Wrestling Manager.exe`.
  - If Windows SmartScreen appears, click "More info", then "Run anyway". This is normal for an unsigned indie build. Please allow up to a minute to load up
  - The game opens in its own window.

  **Linux:**
  - Right-click `H2H-Wrestling-Manager`, go to Properties, Permissions, enable "Allow executing file as program" (or run
  `chmod +x H2H-Wrestling-Manager` in a terminal).
  - Double-click it, or run it from a terminal.
  - The game opens in your default browser. This is expected on Linux, not a bug - it avoids a sandbox restriction that
  several Linux distributions impose on desktop apps.

  No install, no admin rights, nothing written outside the folder you extract it to. Your save is created in a `data` folder
  next to the game.
